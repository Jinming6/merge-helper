# merge-helper

轻松处理单元格的合并

## 特性

- [ ] 跨行合并
- [ ] 跨列合并
- [ ] 自定义条件合并

## 安装

### pnpm

```bash
$ pnpm add merge-helper
```

### cdn

```html
<script src="https://unpkg.com/merge-helper/dist/mergeHelper.min.js"></script>
```

## 快速上手

```js
import MergeHelper from 'merge-helper';

const mergeHelper = new MergeHelper();
```
